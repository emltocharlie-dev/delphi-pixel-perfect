# Delphi Digital像素级复刻 - OKX教程中心

## 项目概述

这是一个像素级复刻Delphi Digital (https://members.delphidigital.io/) 网站的项目，仅替换文字内容为OKX交易教程，不包含会员系统，是一个纯内容网站。

## 项目状态

✅ **已完成** (2026-02-25)
- [x] 深度技术分析 (基于原站完整HTML分析)
- [x] 像素级设计系统实现 (色彩、字体、间距完全匹配)
- [x] 组件与布局实现 (导航、卡片、筛选、搜索)
- [x] 内容管理系统集成 (11篇教程，标签映射)
- [x] 交互功能完整实现 (标签切换、搜索、筛选、分页)

## 技术栈

### 基于深度技术分析的原站技术栈
- **CSS框架**: Tailwind CSS v3.4.1 (内联样式)
- **字体**: NeurialGrotesk (Regular, Medium, Bold变体)
- **JavaScript框架**: Vue.js 3.x (推断)
- **构建工具**: Vite或Webpack

### 本项目实现的技术栈 (简化版)
- **HTML5**: 语义化标签，无障碍支持
- **CSS3**: 自定义属性(CSS变量)，完全匹配原站设计系统
- **Vanilla JavaScript**: 零依赖，原生实现所有交互
- **响应式设计**: 移动端优先，多断点适配

## 设计系统

## 扩展与维护

### 添加新教程
1. 编辑 `data/content.js` 中的 `CONTENT_DATA.articles` 数组
2. 添加新的文章对象，包含完整元数据
3. 分配适当的 `delphiTags` 数组
4. 重新部署网站

### 自定义样式
1. 编辑 `styles/design-system.css` 中的CSS变量
2. 修改色彩、间距、字体等设计系统参数
3. 添加新的工具类或组件样式

### 添加新功能
1. 在 `scripts/app.js` 中添加新功能模块
2. 更新 `ContentManager` API (在 `data/content.js`)
3. 添加对应的UI组件到 `index.html`
